lab1中printfmt函数用到了可变参，请参考写一个小的linux应用程序，完成实现定义和调用一个可变参数的函数。(spoc)

```
#include <stdlib.h>
#include <stdarg.h>
#include <stdio.h>

int min(int nparam, ...) {
    if (nparam <= 0) return 0;
    va_list ap;
    va_start(ap, nparam);
    int i, tmp, min = va_arg(ap, int);
    for (i = 1; i < nparam; i ++)
        if ((tmp = va_arg(ap, int)) < min) min = tmp;
    return min;
}

int main() {
    printf("%d\n", min(3, 999, -2, 4));
}
```

- 以上程序实现了可变参数的函数min，第一个参数是可变参数的数量，函数输出所有可变参数的最小值，如果第一个参数非法 (<= 0)，则返回0.
